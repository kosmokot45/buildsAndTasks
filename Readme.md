## _Запуск и работа с программой_

##### 1. Открыть терминал в корне проекта
##### 2. Активировать виртуальную среду
# 
```sh
.\venv\Scripts\activate
```
##### 3. Для вывода списка задач использовать команду:
# 
```sh
py main.py list tasks
```
##### 4. Для вывода списка билдов использовать команду:
# 
```sh
py main.py list builds
```
##### 5. Для вывода детальной информации по задаче использовать команду:
# 
```sh
py main.py get task your_task
```
например
```sh
py main.py get task upgrade_lime_leprechauns
```
##### 6. Для вывода детальной информации по билду использовать команду:
# 
```sh
py main.py get build your_build
```
например
```sh
py main.py get build time_alone
```
## _Запуск тестов_
##### 1. Для запуска всех тестов использовать команду:
# 
```sh
pytest test.py
```
##### 2. Для запуска определенного теста, например для теста с корректным выводом детальной информации по билду, использовать команду:
# 
```sh
pytest test.py::test_get_data_build
```
##### 3. Для запуска определенного теста, например для теста с некорректным билдом, использовать команду:
# 
```sh
pytest test.py::test_get_data_build_unknow
```
